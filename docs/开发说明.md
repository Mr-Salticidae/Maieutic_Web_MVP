# 开发说明

## 结构

```text
src/
├── app/
│   ├── api/chat/route.ts   # 后端聊天接口
│   ├── globals.css         # Tailwind 全局样式
│   ├── layout.tsx
│   └── page.tsx
├── components/
│   ├── BeaconCard.tsx
│   ├── ChatBox.tsx
│   ├── InsightCard.tsx
│   └── MessageList.tsx
└── lib/
    ├── aiClient.ts         # provider / fallback / API 调用
    ├── maieuticPrompt.ts   # Maieutic 系统提示词
    ├── mockResponse.ts     # 无 Key 时的可测试响应
    ├── parseResponse.ts    # JSON / markdown fallback 解析
    └── types.ts
```

## 模型调用

`aiClient.ts` 支持：

- `AI_PROVIDER=openai`：走 OpenAI-compatible `/chat/completions`。
- `AI_PROVIDER=anthropic`：走 Anthropic `/messages`。
- `FALLBACK_*`：主 provider 失败后尝试 fallback。
- `MAIEUTIC_MOCK=1`：强制 mock。

没有 API Key 时也会自动 mock，保证前端能先跑通。

## 输出解析

系统 prompt 要求模型输出 JSON：

```json
{
  "mode": "Knowledge",
  "reply": "主要回答正文",
  "insight": null,
  "beacon": null
}
```

如果模型输出 markdown，`parseResponse.ts` 会尝试按 `## Insight` / `## Beacon` 做兜底解析。
